### 1. Color Palette (The "Atmosphere")
To maintain the NASA/Space-Age aesthetic, all plugins must strictly adhere to these HEX codes:
* **Background:** #0B0E11 (Deep Space Charcoal) ‚Äî Used for the main plugin window.
* **Primary Text/Lines:** #52D396 (Luminous Mint) ‚Äî Used for labels, delicate borders, and "Safe" data readouts.
* **Button/Active States:** #00B87C (High-Tech Emerald) ‚Äî Used for buttons, active toggles, and "Engaged" parameters.
* **Data Visualization:** Transparent gradients of Mint to represent signal flow and density.

### 2. The "L" Anchor (Developer Menu)
* **Position:** Always fixed to the **Top-Right** corner of every plugin interface.
* **Function:** Tapping the Mint Green "L" logo opens a slide-out "Telemetry" panel.
* **Contents:** About/Version info, External MIDI Sync, and Oversampling toggles.

### 3. Typography & Layout (The "Sonible" Influence)
* **Density:** High-density data displays using ultra-thin lines.
* **Bezel-less Design:** No heavy 3D-rendered knobs or thick borders. Modules should feel like they are floating in the background.
* **Liquid Motion:** All meters must use smooth, non-linear ballistics to mimic fluid physics.

### 4. Inter-Plugin "Link-Sync"
* Each UI must feature a **"Docking Status"** indicator.
* **Glow State:** When the plugin is communicating with **Lumina Control**, the indicator pulses in #52D396.
